# FinTrack-Pro-Expense-Tracker
A modern personal finance tracker built with HTML, CSS and JavaScript featuring authentication, expense tracking, charts, budgeting, dark mode and local storage.

- 🔐 Login & Register
- 💰 Income & Expense Tracking
- ✏️ Edit Transactions
- ❌ Delete Transactions
- 📊 Monthly Analytics (Chart.js)
- 🥧 Expense Breakdown
- 💵 Budget Management
- 🌙 Dark Mode
- 🔍 Search & Filter
- 👤 User Settings
- 💱 Currency Selection
- 🔄 Reset All Data
- 💾 LocalStorage Support
- 📱 Fully Responsive
